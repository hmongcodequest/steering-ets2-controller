You need both the PC Server and the Android Application to establish the connection.

PC Server

The lightweight server application that translates your phone's movements into vJoy commands.

Windows 10/11 Only
vJoy Driver Required
Download PC Server (.exe)

Android App

The controller interface for your phone. Connects to the PC server to transmit steering data.

Android 8.0+ Required
Wi-Fi or USB connection
Download App (.apk)

Installation Instructions

  1. Download and install vJoy on your Windows PC.
  2. Download the PC Server and run it. It may prompt for Firewall access; please allow it.
  3. Download the Android App APK and install it on your smartphone.
  4. Ensure both devices are on the same Wi-Fi network, or connect via USB using ADB reverse port forwarding.
  5. Enter the IP address shown on the PC Server into the Android App and press Connect!
